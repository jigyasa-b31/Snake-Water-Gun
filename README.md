# Snake Water Gun Game

## Description
A menu-driven Snake Water Gun game built using Python.
The user plays against the computer using simple game logic.

## Features
- Menu-driven gameplay
- Random computer choice
- Input validation
- Multiple rounds support
- Win / lose / tie result display

## Technologies Used
- Python

## How to Run
1. Open terminal inside the project folder
2. Run the following command:
   python main.py

## What I Learned
- Implementing game logic using conditional statements
- Working with functions in Python
- Using random module for computer choices
- Taking user input and validating it
- Building a complete beginner-friendly CLI game
